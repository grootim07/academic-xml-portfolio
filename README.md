# Academic XML & Metadata Portfolio

This repository showcases advanced proficiency in structural XML tagging, schema compliance, and metadata engineering for scholarly publishing.

## 📁 Repository Structure
`/samples`: Contains fully formatted, open-source mock files demonstrating metadata processing capabilities.
`metadata-sample.xml`: A clean, deeply nested XML dataset handling complex namespaces, contributor strings, and bibliographic reference parsing.

**Skills Demonstrated**
**Namespace Management:** Proper separation of core schemas and metadata configurations (`xmlns:meta`).
**Granular Contributor Mapping:** Complete isolation of multi-author groups, sequence attributes, and affiliation links.
**Citation & Bibliography Parsing:** Deep tagging of unstructured bibliographic strings into indexable components (authors, titles, years).
